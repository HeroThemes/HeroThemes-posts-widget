== Description ==

A simple widget for displaying posts in widgetized areas of your site, includes support for ordering, exclusion, comments, excerpt and thumbnail display

== Installation ==

It's easy to get started

1. Upload `ht-posts-widget` to the `/wp-content/plugins/` directory or goto Plugins > Add New and search for Heroic Posts Widget.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. In Appearance > Widgets there will be a new Heroic Posts Widget, drag this to the widget area and set the simple options to control number of posts, sort order etc.
